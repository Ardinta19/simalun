# Dokumen Pendukung SIMALUN

Folder ini menyimpan dokumen analisis dan riset yang dipakai tim selama
pengembangan. Dokumen di sini bersifat referensi — bukan bagian dari kode
aplikasi.

## Isi

| Berkas | Keterangan |
|---|---|
| `project-charter.pdf` | Project charter awal: tujuan, ruang lingkup, peran tim, jadwal. |
| `srs-simalun-kelompok2.pdf` | Software Requirements Specification — kebutuhan fungsional & non-fungsional. |
| `wawancara-pemilik.docx` | Transkrip wawancara dengan pemilik Azka Laundry (sumber data primer). |
| `wawancara/wa-*.jpeg` | Tangkapan layar percakapan WhatsApp tambahan saat klarifikasi kebutuhan. |

## Catatan

- Dokumen di sini boleh diganti versi terbarunya selama nama file tetap.
- Untuk file kerja sementara (draf, revisi belum final), pakai folder
  `docs/temp/` — sudah masuk `.gitignore` jadi tidak ikut tertrack.
